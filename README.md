## How to Use This Repo

1. **Weekly log** – create a new `.md` file in `weekly_logs/` each week. Name format: `weekX_YYYY_MM_DD.md`. Log hours daily.
2. **Certificates** – upload screenshots or PDFs to `certificates/`.
3. **Job applications** – update `job_applications/log.md` or an Excel/CSV file.
4. **Monday report** – create a file in `weekly_reports/` named `report_weekX_YYYY-MM-DD.md` and paste the required four items.
5. **Commit often** – at least 5 days per week.
